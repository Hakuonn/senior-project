平台服務條款

歡迎使用本平台！在您註冊、登入及使用我們的網站前，請仔細閱讀以下服務條款，這些條款將規範您使用本平台的權利與義務。當您使用本平台時，即表示您同意接受以下條款的約束。

1. 平台使用

本平台旨在提供剩食訂購與健康食譜服務，用戶可以透過平台瀏覽、訂購剩食以及查看相關的健康食譜。
平台所提供的資訊僅供參考，平台不對任何健康建議負責，建議用戶在訂購前自行判斷及諮詢專業意見。

2. 用戶註冊

用戶需註冊帳號才能使用本平台的完整功能。用戶需提供準確的個人資訊，並對其帳號活動負全責。
用戶應妥善保管帳號密碼，不得將帳號提供給第三方使用。

3. 訂購與支付

用戶可透過本平台訂購剩食，並需於訂購時完成支付。所有訂購將依照當前平台顯示的價格進行。
平台將盡力確保訂單準確無誤，但在某些情況下，可能會因庫存不足或其他因素導致訂單無法完成，平台將通知用戶並提供退款或其他解決方案。

4. 健康食譜

本平台所提供的健康食譜內容來自專業的營養師或其他授權提供者，旨在為用戶提供健康飲食建議。
用戶應根據自身健康狀況，諮詢醫師或營養師的建議後，決定是否採用該食譜。

5. 責任限制

平台對於剩食品質、食譜準確性或適用性不承擔任何法律責任。所有資訊及服務均以「現狀」提供，平台不對其適用性、可靠性或可用性作出任何承諾。
平台不對因用戶使用本平台服務或無法使用本平台服務所產生的任何直接或間接損失負責。

6. 用戶責任

用戶同意不得使用本平台進行任何違法行為，包括但不限於欺詐、侵犯他人權利等行為。
用戶在使用平台服務時，需遵守所有適用法律及本條款規範。

**隱私政策**

本平台非常重視您的隱私權，並致力於保護您的個人資訊。本隱私政策將詳細說明我們如何收集、使用、儲存及分享您的個人資料，並告知您在隱私保護方面的權利及選擇。當您使用本平台服務時，即表示您同意我們依據本政策收集和使用您的個人資料。

### 1. **我們收集的資訊**

在您使用我們的平台時，我們可能會收集以下類型的個人資料：

**1.1 用戶註冊資訊**  
- 當您註冊帳戶時，我們會收集您提供的資訊，包括但不限於您的姓名、電子郵件地址、電話號碼、居住地等資料。

**1.2 訂單與支付資訊**  
- 在您進行剩食訂購時，我們會收集與交易相關的資訊，例如購買紀錄、支付方式（如信用卡或第三方支付帳號）、送貨地址等資訊。

**1.3 平台互動數據**  
- 我們可能會自動收集您與平台互動的數據，包括您的裝置資訊、瀏覽行為（如瀏覽的頁面、點擊的連結）、IP 位址、裝置識別碼等技術資訊。

**1.4 健康資訊（可選）**  
- 若您選擇提供與健康有關的資料，例如過敏資訊或飲食偏好，我們會將這些資訊用於提供個性化的健康食譜建議。

### 2. **我們如何使用您的資訊**

我們會根據合法的業務需求及您的同意來處理和使用您的個人資訊，具體用途包括：

**2.1 平台運營與管理**  
- 使用您的註冊資料來創建及管理帳戶，並為您提供平台上的各種服務。

**2.2 訂單處理與履行**  
- 使用您的訂單資訊來完成交易，安排送貨，並聯絡您以處理相關事宜。

**2.3 個性化推薦與服務**  
- 根據您的偏好和歷史數據，為您推薦個性化的健康食譜，並提供相應的剩食訂購建議。

**2.4 通訊與行銷活動**  
- 向您發送平台更新、促銷活動、優惠資訊等，前提是您已同意接收此類通訊。

**2.5 資料分析與改進**  
- 收集的技術數據可幫助我們進行平台的維護、功能改進及用戶體驗的優化。

### 3. **我們如何分享您的資訊**

我們承諾不會出售或出租您的個人資料，除以下情況外，我們不會與第三方分享您的資訊：

**3.1 服務提供商**  
- 我們可能會將您的個人資料分享給協助我們運營的平台合作夥伴，如支付處理、物流配送、數據分析等第三方服務提供商。他們只會基於我們的指示處理您的資訊，並遵守本隱私政策的規範。

**3.2 法律要求**  
- 在法律要求或應政府機構的合法請求下，我們可能會披露您的個人資訊。這包括但不限於依據法院命令、執法機構調查或其他法律程序要求。

**3.3 業務變更**  
- 在公司進行重組、合併或出售部分或全部資產的情況下，您的個人資訊可能作為資產之一轉讓給第三方，但我們會確保受讓方承擔保護您的個人資料的責任。

### 4. **資料儲存與安全**

我們採取合理的技術和組織措施來保護您的個人資料不受未經授權的存取、使用或洩露。具體措施包括：

- 使用加密技術保護支付資訊；
- 限制對個人資料的存取，僅授權的員工或合作夥伴可訪問；
- 定期審查我們的安全流程及技術，防止資料外洩。

然而，儘管我們已採取合理的保護措施，互聯網上的傳輸並非完全安全，我們無法保證您的資料在傳輸過程中的絕對安全性。

### 5. **您的權利**

根據相關隱私法規，您擁有以下權利：

**5.1 查閱與更新**  
- 您有權查閱我們所持有的個人資料，並可以隨時登錄您的帳戶更新或更正您的個人資訊。

**5.2 刪除與撤回同意**  
- 您有權請求刪除您的個人資料，或在您已同意的情況下隨時撤回同意。我們將在合理的時間內處理您的請求，並停止處理您的個人資料，除非有合法的原因繼續保留。

**5.3 資料轉移**  
- 您有權獲得我們收集的您的個人資訊副本，並可以選擇將其傳輸給另一個服務提供商。

### 6. **Cookie 與追蹤技術**

我們可能會使用 Cookie 及類似的技術來收集您在使用平台時的瀏覽行為數據，以便於優化您的體驗。您可以通過瀏覽器設置控制或禁用 Cookie，但這可能會影響某些功能的使用。

### 7. **隱私政策的變更**

我們可能會不定期更新本隱私政策，以反映我們的業務調整或法律法規的變更。政策更新後，我們將會通知您，並將變更發布於平台上。建議您定期查看本政策，以了解最新的隱私保護規定。
